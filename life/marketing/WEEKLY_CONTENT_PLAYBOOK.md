## Weekly Trigger
Every Monday, Josh sends: "Run weekly content batch — Week [X]"

## What to produce each week

### Instagram (5 posts)
Post 1–3: Faceless — tip, quote, or carousel idea
 - Write caption (150–200 words + 5 hashtags)
 - Generate a DALL-E image: warm, illustrated, cozy homeschool vibe, no real people
 - Save image to: life/marketing/instagram/week-[X]/post-[N]-image.png
 - Save caption to: life/marketing/instagram/week-[X]/post-[N]-caption.txt

Post 4–5: Personal — Josh's face/story
 - Write caption (personal, first-person, story-driven, 150–200 words + 5 hashtags)
 - Write a photo direction note: exactly what Josh should photograph or film
 - Save to: life/marketing/instagram/week-[X]/post-[N]-caption.txt
 - Save photo direction to: life/marketing/instagram/week-[X]/post-[N]-photo-direction.txt

### Substack (1 article)
 - Topic: rotate through pillars (see below)
 - Length: 800–1200 words
 - Format: Short paragraphs, no jargon, personal opening story, practical takeaways
 - Save to: life/marketing/substack/week-[X]-article.md

### Weekly summary
 - Save to: life/marketing/week-[X]-summary.md
 - List all files created and a one-line description of each
 - Include Josh's photo directions for personal posts
 - Include Substack article title and first paragraph
