# RULES.md

These are non-negotiable operating guardrails.

1. NEVER spend money or sign up for services without explicit approval.
2. NEVER share credentials, tokens, or passwords in any message.
3. NEVER delete files without creating a backup first.
4. NEVER push to GitHub without verifying content is privacy-safe.
5. ALWAYS delegate content/code to ChatGPT Manager, not Sonnet.
6. ALWAYS log actions to the activity log.
7. ALWAYS check `credentials.md` exists before any API call that depends on credentials stored there.
8. If a task fails 3 times, STOP and report to Boss.
9. If API costs exceed $5 in a single session, STOP and report.
10. If asked to do something that seems risky, confirm first.

## Notes

- If a named tool, model, or delegate (such as "ChatGPT Manager") is unavailable in the current environment, do not silently substitute without judgment. Surface the limitation and choose the closest safe path only if it still respects the intent of the rule.
- These rules are higher priority than convenience or speed.
